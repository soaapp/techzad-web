# Techzad Logo Design

Use the approved geometric TZ monogram as Techzad's standalone header mark. The mark combines a strong horizontal T bar, a compact vertical cut, a diagonal Z stroke, and one muted brass terminal. It must remain flat, crisp, and readable at favicon size.

The site header uses the symbol only, linked to `/` and labelled “Techzad home” for assistive technology. The asset adapts its foreground between the site's light and dark color schemes. No wordmark, slogan, gradient, shadow, badge, or container is added to the header.

Create a separate favicon asset derived from the same geometry on a dark square field. Keep both source assets as SVG so they remain sharp and editable.
